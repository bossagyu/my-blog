## こんな悩みはありませんか？

- 「掃除機、最後にかけたのいつだっけ...」
- 「トイレ掃除、そろそろやらないと」
- 「換気扇の掃除、忘れてた！」

掃除は「やらなきゃ」と思っていても、つい忘れがち。特に週1回、月1回といった定期的な掃除は、気づいたら期限を過ぎていることが多いですよね。

## 掃除リマインダーBotを作りました

そんな悩みを解決するために、**掃除リマインダーLINE Bot**を作りました。

このBotは、掃除タスクごとに「何日ごとにやるか」を設定でき、期限が来たらLINEで通知してくれます。

### 主な機能

- **タスク登録**: 「掃除機 7日」「トイレ 3日」など、タスクと間隔を設定
- **完了報告**: 掃除が終わったらLINEで報告するだけ
- **リマインド通知**: 設定した曜日・時間に期限切れタスクを通知
- **一時停止**: 旅行中などはタスクを停止できる

## 使い方

### 1. 友だち追加

以下のQRコードまたはリンクから友だち追加してください。

![QRコード](qrcode.png)

[友だち追加はこちら](https://lin.ee/kdwsLys)

### 2. タスクを追加する

```
追加 掃除機 7
```

これで「7日ごとに掃除機」というタスクが登録されます。

### 3. 掃除が終わったら報告

```
完了 掃除機
```

タスクの期限がリセットされ、次の通知まで7日間になります。

### 4. 残りのタスクを確認

```
残り
```

期限切れのタスクが一覧で表示されます。何日超過しているかも分かります。

```
タスクは以下の通りです。
掃除機 (+3日)
トイレ (+1日)
```

### 5. 通知設定

好きな曜日と時間に通知を受け取れます。

```
通知設定 月水金 7
```

これで月・水・金の7時に通知が届きます。

## なぜ「日数経過」で管理するのか

一般的なTODOアプリは「期日」を設定しますが、掃除は「前回からどれくらい経ったか」で判断することが多いです。

- 掃除機は「1週間に1回くらい」
- トイレは「3日に1回」
- 換気扇は「月に1回」

このBotは、完了報告をするたびに自動で次の期限が設定されるので、毎回日付を入力する手間がありません。
